# Agent Feedback Refactor Implementation Plan

> **Legacy execution note:** This imported plan was originally written for the Superpowers workflow. Use the surrounding GSD quick-task artifact and current repository instructions for any future execution; the original checkbox steps remain for historical traceability.

**Goal:** Split `agent_feedback.py` into focused modules without changing the public agent-feedback sidecar API or JSON output.

**Architecture:** Keep `src/sol_execbench/core/bench/agent_feedback.py` as the public facade for builders, artifact citation, freshness validation, and governance evaluation. Move agent-feedback enums and Pydantic sidecar models into `agent_feedback_models.py`; move feedback-item derivation from evaluation statuses into `agent_feedback_items.py`. The facade imports and re-exports moved names so existing callers remain compatible.

**Tech Stack:** Python 3.12+, Pydantic v2, pytest, Ruff, existing SOL ExecBench ROCm `Trace`, `EvaluationStatus`, `Rocprofv3ProfileResult`, and `StaticKernelEvidenceSidecar` types.

---

## File Structure

- Create `src/sol_execbench/core/bench/agent_feedback_models.py`
  - Owns agent-feedback enums and Pydantic models.
  - Exports `AGENT_FEEDBACK_SCHEMA_VERSION`, `AgentFeedbackStatus`, `AgentFeedbackReasonCode`, `AgentFeedbackSeverity`, `AgentFeedbackFreshnessStatus`, `AgentFeedbackGovernanceStatus`, `AgentFeedbackBottleneck`, `AgentFeedbackSourceRef`, `AgentFeedbackArtifactCitation`, `AgentFeedbackIdentity`, `AgentFeedbackFreshnessValidation`, `AgentFeedbackGovernanceGuardrail`, `AgentFeedbackItem`, `AgentFeedbackSummary`, and `AgentFeedbackSidecar`.
- Create `src/sol_execbench/core/bench/agent_feedback_items.py`
  - Owns status-to-feedback-item mapping.
  - Exports `trace_feedback_items(status_counter: Counter[EvaluationStatus])`.
- Modify `src/sol_execbench/core/bench/agent_feedback.py`
  - Keep `build_agent_feedback_sidecar()`, `artifact_citation_from_path()`, `validate_agent_feedback_freshness()`, `evaluate_agent_feedback_governance()`, `_aggregate_status()`, `_source_refs()`, and `_limitations()`.
  - Import and re-export moved model/enumeration names.
  - Replace `_trace_feedback_items(status_counter)` with `trace_feedback_items(status_counter)`.
  - Remove local definitions for the moved models/enums and item-derivation helpers.
- Create `tests/sol_execbench/test_agent_feedback_models.py`
  - Locks down facade re-export compatibility and strict/frozen validation behavior after model extraction.
- Create `tests/sol_execbench/test_agent_feedback_items.py`
  - Locks down feedback item derivation independently from the sidecar builder.
- Modify `tests/sol_execbench/test_agent_feedback.py`
  - Keep public facade tests for identity, sidecar builder, artifact citations, freshness, governance, authority guardrails, claim upgrade boundaries, and one end-to-end item integration check.
  - Remove or avoid duplicating item-mapping-only assertions now covered in `test_agent_feedback_items.py`.

from __future__ import annotations

from collections import Counter

from sol_execbench.core.bench.agent_feedback_items import trace_feedback_items
from sol_execbench.core.data.trace import EvaluationStatus


def test_trace_feedback_items_reports_all_passed_when_no_failure_items() -> None:
    items = trace_feedback_items(Counter({EvaluationStatus.PASSED: 2}))
    payload = [item.model_dump(mode="json") for item in items]

    assert payload == [
        {
            "code": "all_evaluated_traces_passed",
            "severity": "info",
            "bottleneck": "unknown",
            "message": "All evaluated traces passed; no failure-specific diagnostic is available.",
            "recommendation": "Use optional profiling or static evidence for next-step performance diagnosis.",
            "source_refs": [
                {
                    "kind": "trace",
                    "label": "canonical_trace_jsonl",
                    "status": None,
                }
            ],
        }
    ]


def test_trace_feedback_items_reports_compile_runtime_timeout_and_policy_failures() -> None:
    items = trace_feedback_items(
        Counter(
            {
                EvaluationStatus.COMPILE_ERROR: 1,
                EvaluationStatus.RUNTIME_ERROR: 2,
                EvaluationStatus.TIMEOUT: 3,
                EvaluationStatus.REWARD_HACK: 4,
            }
        )
    )

    assert [(item.code, item.severity.value, item.bottleneck.value) for item in items] == [
        ("compile_error", "action", "compile_failure"),
        ("reward_hack", "action", "policy_violation"),
        ("runtime_error", "action", "runtime_failure"),
        ("timeout", "action", "timeout"),
    ]
    assert all(item.source_refs[0].label == "canonical_trace_jsonl" for item in items)


def test_trace_feedback_items_reports_correctness_and_reference_failures() -> None:
    items = trace_feedback_items(
        Counter(
            {
                EvaluationStatus.INCORRECT_NUMERICAL: 1,
                EvaluationStatus.INCORRECT_SHAPE: 1,
                EvaluationStatus.INCORRECT_DTYPE: 1,
                EvaluationStatus.INVALID_REFERENCE: 1,
            }
        )
    )

    assert [(item.code, item.severity.value, item.bottleneck.value) for item in items] == [
        ("INCORRECT_DTYPE", "action", "interface_correctness"),
        ("incorrect_numerical", "action", "numerical_correctness"),
        ("INCORRECT_SHAPE", "action", "interface_correctness"),
        ("invalid_reference", "warning", "reference_failure"),
    ]
