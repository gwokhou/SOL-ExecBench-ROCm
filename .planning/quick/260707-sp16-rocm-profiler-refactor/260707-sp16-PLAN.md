# ROCm Profiler Refactor Implementation Plan

> **Legacy execution note:** This imported plan was originally written for the Superpowers workflow. Use the surrounding GSD quick-task artifact and current repository instructions for any future execution; the original checkbox steps remain for historical traceability.

**Goal:** Split `rocm_profiler.py` into focused model, command, artifact, profile collection, and timing modules while preserving the existing public import surface.

**Architecture:** Keep `sol_execbench.core.bench.rocm_profiler` as the public facade. Move dataclasses/constants to `rocm_profiler_models.py`, command construction and default runners to `rocm_profiler_commands.py`, artifact discovery/classification to `rocm_profiler_artifacts.py`, optional profile artifact collection to `rocm_profiler_profile.py`, and CSV/timing collection to `rocm_profiler_timing.py`.

**Tech Stack:** Python 3.12 dataclasses, pytest, Ruff, existing `TimingPolicy`/`TimingBackend` APIs.
