# Roadmap: v1.14 Optional rocprofv3 Profiling Evidence

**Defined:** 2026-05-25
**Status:** ACTIVE
**Source:** `.planning/notes/2026-05-23-rocm-systems-gpuopen-enhancements.md`

## Milestone Goal

Make `rocprofv3` profiling evidence available as an explicit diagnostic layer
for benchmark runs, with durable artifact metadata and no default behavior
change.

## Phase Structure

### Phase 61: Profiling Option and Command Provenance

Goal: introduce the opt-in CLI/config surface and profiler command builder.

Scope:

- Add `--profile rocprofv3` or equivalent profile selection.
- Validate profiler availability and output directory setup.
- Record command provenance and skipped/unavailable states.

Acceptance:

- Default benchmark commands do not invoke `rocprofv3`.
- Missing profiler tools produce clear diagnostic metadata.

### Phase 62: rocprofv3 Artifact Lifecycle

Goal: collect and register profiler artifacts in stable result metadata.

Scope:

- Prefer `rocpd` output when supported.
- Register CSV trace/counter/agent-info outputs where available.
- Preserve profiler exit status and stdout/stderr tails.
- Add artifact cleanup/retention behavior consistent with existing run output.

Acceptance:

- Enabled profiling records artifact paths and collection status.
- Profiler failure is visible but non-fatal to correctness evaluation.

### Phase 63: Profiling Reports and Documentation

Goal: make profiling evidence understandable and operationally usable.

Scope:

- Add reporting surfaces for profiler status and artifact locations.
- Document Docker/local requirements, permissions, and troubleshooting.
- Add tests for success, skipped, unavailable, and failed profile states.

Acceptance:

- Users can enable profiling and locate artifacts from result metadata.
- Documentation distinguishes profiling evidence from score authority.

## Compatibility Notes

v1.14 should remain non-breaking unless a future explicit decision makes
profiling evidence a required consumer policy. That decision is out of scope for
this planned milestone.
