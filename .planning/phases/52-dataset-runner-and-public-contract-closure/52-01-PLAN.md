| Local filesystem to runner | User-supplied sidecar and trace JSON files are read from disk. |
| Runner to derived report | Internal SOLAR evidence influences report warnings and score eligibility. |
| Derived report to public consumers | Report wording and keys can imply unsupported claims if not scoped. |

| T-52-01-02 | Repudiation | skipped-trace report path | mitigate | Add regression tests proving skipped traces produce requested sidecars/report rows and failed traces still rerun. |
| T-52-01-03 | Information Disclosure | canonical trace JSONL | mitigate | Keep all SOLAR fields in sidecars or derived-report-only metadata; do not write them into canonical `traces.json`. |
