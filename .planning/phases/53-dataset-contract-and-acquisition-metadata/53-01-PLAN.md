<objective>
Add a reusable dataset contract library that validates the public dataset
layout and emits deterministic acquisition/local-layout manifests without
touching public benchmark schemas or execution paths.
</objective>

<threat_model>
- Path traversal via category or manifest paths: validate categories against
  fixed constants and keep manifest serialization deterministic.
- Credential leakage: manifest source fields must not record Hugging Face
  tokens, environment variables, or auth headers.
- Claim overreach: manifest metadata must explicitly mark ROCm readiness,
  execution success, paper validation, hosted leaderboard parity, and upstream
  SOLAR equivalence as false.
</threat_model>
