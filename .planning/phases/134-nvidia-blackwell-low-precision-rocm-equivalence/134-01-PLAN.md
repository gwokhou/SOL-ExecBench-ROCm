# Phase 134 Plan: NVIDIA/Blackwell Low-Precision ROCm Equivalence

## Goal

Implement CPU-safe ROCm-compatible low-precision abstractions for migrated
NVIDIA/Blackwell dataset semantics while preserving explicit unvalidated-CDNA4
claim boundaries.

     `float4_e2m1`, `float4_e2m1fn_x2`).
   - Provide deterministic E2M1 quantize/dequantize and two-values-per-byte

     compatibility abstractions are present.

     layers marked as needed.

   - Semantic round trips for pack/unpack/dequantize with known E2M1 codes.
