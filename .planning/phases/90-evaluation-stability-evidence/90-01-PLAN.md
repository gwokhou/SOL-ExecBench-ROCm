# Phase 90 Plan 01: Stability Report Contract

## Goal

Implement strict `sol_execbench.evaluation_stability.v1` models and CPU-safe
classification helpers for timing quality diagnostics.

## Tasks

1. Add `src/sol_execbench/core/evaluation_stability.py`.
2. Support source refs/checksums, deterministic report checksum, JSON output,
   Markdown rendering, and write helpers.
3. Classify timing evidence into stable, noisy, insufficient-samples,
   missing-timing, clock-unlocked, profiler-overhead-risk, and
   backend-unsupported states.
4. Compute deterministic dispersion metrics from existing timing evidence:
   count, min, max, mean, median, population standard deviation, coefficient
   of variation, selected runtime, and selected statistic.
5. Keep explicit false authority boundaries in both JSON and Markdown.
