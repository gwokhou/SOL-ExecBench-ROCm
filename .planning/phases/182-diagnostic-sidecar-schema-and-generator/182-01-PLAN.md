# Phase 182 Plan 01: Agent Feedback Sidecar Schema

**Phase:** 182 - Diagnostic Sidecar Schema and Generator
**Status:** Complete
**Requirements:** SIDE-01, SIDE-03, SIDE-04

## Goal

Define a strict diagnostic-only `sol_execbench.agent_feedback.v1` schema and
builder that summarizes existing trace/profile/static evidence into bounded
feedback metadata.

## Tasks

1. Add `src/sol_execbench/core/bench/agent_feedback.py`.

3. Add builder logic for trace status summaries and optional diagnostic input

4. Add CPU-safe schema tests.
