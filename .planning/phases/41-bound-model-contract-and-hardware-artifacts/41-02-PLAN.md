| T-41-07 | Information Disclosure | score evidence refs | accept | Evidence refs are explicit local paths/labels already emitted by derived reports; this plan preserves existing behavior and does not add secrets. |

| T-41-SC | Tampering | package installs | accept | No package installs are planned; this plan uses existing code and tests only. |
