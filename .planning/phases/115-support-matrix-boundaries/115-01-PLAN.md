<threat_model>
## Trust Boundaries

| Boundary | Description |
| --- | --- |
| Documentation -> user interpretation | Public docs can cause users to overstate support or validation claims if wording is ambiguous. |
| Container evidence -> native-host claims | Container user-space evidence can be incorrectly upgraded to native-host validation. |
| Readiness metadata -> hardware validation | MI300X/CDNA3 readiness can be incorrectly read as completed validation. |
