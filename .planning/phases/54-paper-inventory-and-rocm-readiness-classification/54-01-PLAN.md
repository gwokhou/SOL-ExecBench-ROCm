<objective>
Implement the deterministic dataset inventory core that converts a dataset root
into sidecar metadata using canonical Pydantic contracts.
</objective>

<threat_model>
- Untrusted dataset paths: store relative references from the selected dataset
  root and do not write into canonical dataset files.
- Schema confusion: use `Definition` and `Workload` as the only public contract
  parsers, not ad hoc key checks.
- Claim overreach: inventory records metadata only and must not include
  readiness/pass/score/paper-parity claims.
</threat_model>
