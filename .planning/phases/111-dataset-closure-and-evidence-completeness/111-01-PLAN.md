# Phase 111 Plan: Centralize Dataset Closure And Evidence Completeness Helpers

## Goal

Dataset closure records for selected workloads should be assembled through
tested package helpers with explicit incomplete-state semantics.

## Tasks

1. Add a package-level helper in `run_closure.py` that combines trace status,
   trace/summary/solution refs, derived evidence refs, and evidence gaps into a
   serialized closure record.
2. Update `scripts/run_dataset.py` to delegate selected-workload attempted and
   skipped closure records to that helper.
3. Add focused tests proving the helper marks missing traces and missing
   requested evidence deterministically.
4. Run focused dataset closure tests and Ruff on touched files.

## Acceptance Criteria

- Missing selected workload traces classify as `missing_trace`.
- Requested-but-missing timing/derived sidecars classify as
  `derived_evidence_missing`.
- Reuse and fresh execution paths use the same helper.
- Public execution-closure sidecar schema remains unchanged.
