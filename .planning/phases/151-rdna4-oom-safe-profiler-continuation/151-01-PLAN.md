# Phase 151 Plan

## Goal

Continue RDNA4 profiler timing replacement without letting known OOM or
long-running targets repeatedly kill Codex or the user shell.

## Tasks

1. Add batch-runner controls for explicit skip lists and a configurable staging
   temp root outside `/tmp`.
2. Add a blocked-sidecar mode for known OOM or timeout targets so the coverage
   ledger records `profiler_blocked` instead of leaving them as plain fallback.
3. Add CPU-safe tests for skip selection, temp-root staging, and blocked
   sidecar resume semantics.
4. Recompute coverage after the new controls, then run only bounded real GPU
   continuations.

## Claim Boundary

This phase improves evidence accounting and runner safety only. It does not
upgrade RDNA4 benchmark, paper parity, score authority, or leaderboard claims.
