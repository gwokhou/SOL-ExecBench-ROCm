# 97-02 Plan: Static Evidence Extractor Status Helpers

Extract static extractor aggregate status/reason classification into a helper so
partial, unavailable, failed, timeout, and collected states can be tested without
running toolchain extractors.

   reason decisions.
3. Add focused tests for collected, partial, unavailable, failed, and timeout
   aggregate states.
4. Run static evidence tests and Ruff on touched files.
