# Research preview interpretation

V3 is a research implementation of the SOL ExecBench/SOLAR responsibility
model for AMD ROCm. Canonical Trace JSONL can support a scoped local evaluation
claim when accompanied by its recorded environment. SOLAR manifests can support
a scoped formal-analysis claim only when all fail-closed stages pass.

Neither surface currently establishes:

- full paper-denominator parity;
- upstream SOLAR output parity;
- NVIDIA B200 equivalence;
- hosted leaderboard authority;
- official release scoring;
- multi-tenant sandbox security;
- validation of hardware not named by the attached evidence.

Profiler, static, decision, profile-summary, agent-feedback and environment
sidecars are diagnostic. The current corpus manifest records official scoring
as unavailable, and the packaged formal architecture profile may reject
analysis when its required audit evidence is absent.

For reproducible review, retain the exact definition/workload/solution inputs,
canonical traces, environment identity, clock/timing policy and any SOLAR
manifest/artifact hashes. Do not promote a local ratio or caller-authored JSON
into release authority.
